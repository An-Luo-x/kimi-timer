# 版本变更记录

所有版本变更按时间倒序排列。

## [V1.0.0] — 2025-06-29

### 新增

- **基础框架搭建**
  - 初始化 npm 项目，配置 `package.json`
  - 配置国内 Electron 镜像源（`npmmirror.com`）
  - 安装 Electron 42.5.0 作为开发依赖
  - 编写最基础的 `main.js` + `index.html` Hello World 骨架

- **核心计时功能**
  - 实现两种计时模式：倒计时（Countdown）和正计时（Stopwatch）
  - 时间显示格式统一为 `HH:MM:SS`，大字号清晰呈现
  - 采用**时间戳差值（Timestamp Difference）**算法实现精准计时
  - 记录启动时间戳 `Date.now()`，通过实时差值计算经过时间
  - 应用窗口最小化或后台运行时，计时精度不受影响
  - 更新频率设置为 100ms，平衡性能与显示流畅度

- **窗口交互**
  - 窗口尺寸固定为 400 × 500 像素，不可调整/最大化
  - 隐藏默认顶部菜单栏（File, Edit...），界面纯净
  - 实现「窗口置顶」功能，提供顶部切换按钮并联动 Electron API
  - 模式切换按钮在倒计时与正计时之间无缝切换

- **提醒系统**
  - 倒计时结束时使用 Web Audio API 生成三段递进式蜂鸣声（880Hz → 1100Hz → 1320Hz）
  - 调用 Windows 系统原生通知（Electron `Notification` API）
  - 界面底色闪烁提醒（CSS 动画，持续 3 秒）
  - 视觉、听觉、系统通知三重提醒，确保用户不会错过

- **界面设计**
  - 极简深色渐变风格（`#1a1a2e` → `#16213e`）
  - 渐变按钮配色（开始：青蓝，暂停：橙黄，重置：红粉）
  - 输入框限制范围（时：0-99，分/秒：0-59）
  - 状态栏实时反馈当前操作状态
  - 键盘 Enter 键快速触发「开始」

- **打包与发布**
  - 安装 `electron-builder` 作为开发依赖
  - 配置 Windows 平台打包参数（`appId`、`productName`、`portable` target）
  - 成功构建 `Kimi Timer 1.0.0.exe` 便携版单文件
  - 输出目录：`dist/` 与 `dist/win-unpacked/`

- **项目文档**
  - 创建 `document/` 目录
  - 编写 8 份项目文档：README、CHANGELOG、技术架构、需求规格、构建打包指南、版本发布说明、开发环境搭建指南、待办清单与路线图
