# 版本发布说明（V1.0.0）

**应用名称：** Kimi Timer  
**版本号：** V1.0.0  
**发布日期：** 2025 年 6 月 29 日  
**发布状态：** 🚀 正式发布  
**适用平台：** Windows 10 / Windows 11（64 位）

---

## 版本概述

Kimi Timer V1.0.0 是桌面计时器的首个正式版本，提供完整的倒计时、正计时、窗口置顶、精准计时和多重提醒功能。采用极简设计风格，以单文件便携版形式发布，无需安装即可使用。

---

## 功能列表

### 核心计时
- ✅ 倒计时模式：自定义时/分/秒，精确倒计时
- ✅ 正计时模式：从 0 开始累计计时，无上限
- ✅ 两种模式一键切换，切换时自动重置
- ✅ 时间显示格式：HH:MM:SS，大字号清晰易读

### 精准计时（V1.0 核心亮点）
- ✅ 采用时间戳差值算法（Timestamp Difference）
- ✅ 记录启动时间戳 `Date.now()`，基于实时差值计算
- ✅ 窗口最小化、后台运行、系统休眠恢复后计时绝对精准
- ✅ 100ms 更新频率，平衡性能与流畅度
- ✅ 支持暂停/恢复，暂停状态精确保存

### 提醒系统
- ✅ Web Audio API 生成三段递进式蜂鸣声（880Hz → 1100Hz → 1320Hz）
- ✅ Windows 系统原生通知弹窗
- ✅ 界面底色闪烁视觉提醒（3 秒 CSS 动画）
- ✅ 声音 + 通知 + 视觉三重提醒，确保用户感知
- ✅ 一键重置即可停止所有提醒效果

### 界面设计
- ✅ 极简深色渐变风格（400×500 固定窗口）
- ✅ 隐藏默认顶部菜单栏，界面纯净
- ✅ 窗口置顶切换按钮（带动画反馈）
- ✅ 渐变圆形控制按钮（开始/暂停/重置）
- ✅ 实时状态栏提示
- ✅ 键盘 Enter 键快捷启动

### 打包与部署
- ✅ 单文件便携版 `.exe`（约 87 MB）
- ✅ 无需安装，双击运行
- ✅ 国内镜像源配置，下载稳定
- ✅ 完整项目文档（8 份技术文档）

## 安装方式

### 方式一：便携版（推荐）

1. 下载 `Kimi Timer 1.0.0.exe`
2. 复制到任意文件夹（如 `D:\Tools\`）
3. 双击运行即可

### 方式二：开发环境运行

```bash
git clone <项目仓库>
cd kimi-timer
npm install
npm start
```
