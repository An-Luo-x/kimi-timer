# 待办清单与路线图

本文档记录 Kimi Timer 的已知问题、V1.1 规划功能及长期优化方向。

---

## 当前版本已知问题（V1.0.0）

| 优先级 | 问题 | 状态 | 说明 |
|--------|------|------|------|
| P2 | 打包时需手动处理 PowerShell 路径（Git Bash） | 🔵 记录 | 已有 workaround：使用 CMD 或补充 PATH |
| P2 | 应用使用默认 Electron 图标 | 🔵 记录 | 需创建 `build/icon.ico` 并配置 |
| P3 | 无系统托盘（System Tray）最小化 | 🔵 记录 | 最小化后仍在任务栏，无法收至托盘 |
| P3 | 无多语言支持 | 🔵 记录 | 目前仅中文界面 |
| P3 | 提醒音效不可自定义 | 🔵 记录 | 仅内置蜂鸣声，无法上传自定义音频 |
| P3 | 无自动更新机制 | 🔵 记录 | 需手动下载新版本覆盖 |
| P4 | 无计时历史记录 | 🔵 记录 | 每次关闭应用后数据丢失 |
| P4 | 无键盘快捷键（全局热键） | 🔵 记录 | 如 Ctrl+Alt+T 快速启动/暂停 |
| P4 | 无自定义主题/配色 | 🔵 记录 | 目前仅深色渐变主题 |
| P4 | 无多实例/多标签计时 | 🔵 记录 | 只能同时运行一个计时器 |
| P4 | 无 Pomodoro（番茄工作法）预设 | 🔵 记录 | 常见需求：25分钟/5分钟循环 |
| P4 | 无数据导出功能 | 🔵 记录 | 如 CSV/JSON 导出计时记录 |

## V1.1 规划（短期目标，预计 1-2 个月）

### 新增功能

| 编号 | 功能 | 优先级 | 说明 |
|------|------|--------|------|
| V1.1-01 | 系统托盘（System Tray） | 🔴 高 | 最小化到托盘，右键菜单（开始/暂停/重置/退出） |
| V1.1-02 | 自定义应用图标 | 🔴 高 | 设计并配置 `build/icon.ico` |
| V1.1-03 | 自定义提醒音效 | 🟡 中 | 支持选择本地音频文件作为提醒音 |
| V1.1-04 | 番茄工作法模式 | 🟡 中 | 预设 25min 工作 + 5min 休息循环 |
| V1.1-05 | 计时完成自动循环 | 🟡 中 | 倒计时结束后自动重新开始（可选） |
| V1.1-06 | 窗口位置记忆 | 🟡 中 | 关闭时保存窗口位置，下次启动恢复 |

### 优化项

| 编号 | 优化 | 优先级 | 说明 |
|------|------|--------|------|
| V1.1-OPT-01 | 打包体验优化 | 🟡 中 | 解决 Git Bash 下 PowerShell 路径问题，无需 workaround |
| V1.1-OPT-02 | 代码结构优化 | 🟡 中 | 将 `index.html` 中的 JS 逻辑拆分至独立 `renderer.js` |
| V1.1-OPT-03 | 启用 contextIsolation | 🟢 低 | 提升安全性，使用 `preload.js` 桥接 IPC |
| V1.1-OPT-04 | 输入框体验优化 | 🟢 低 | 支持滚轮调整时间，增加 +/- 按钮 |

## 技术债务

| 编号 | 描述 | 影响 | 计划处理版本 |
|------|------|------|------------|
| TD-01 | `nodeIntegration: true` + `contextIsolation: false` | 安全隐患 | V1.1（通过 preload 脚本重构） |
| TD-02 | `index.html` 中 JS/CSS 全部内联 | 可维护性差 | V1.1（拆分至独立文件） |
| TD-03 | 无单元测试覆盖 | 回归风险 | V1.2（引入测试框架） |
| TD-04 | 无 CI/CD 流水线 | 发布流程手动 | V1.2（配置 GitHub Actions） |
| TD-05 | 打包体积未优化 | 87MB 可压缩 | V1.1（精简 files 白名单） |

## 优先级定义

| 标识 | 含义 | 响应时间 |
|------|------|---------|
| 🔴 P1（高） | 核心功能缺陷或阻塞问题 | 立即处理 |
| 🟡 P2（中） | 影响体验的重要功能 | 当前版本或下一版本 |
| 🔵 P3（低） | 体验优化或nice-to-have | 排入路线图 |
| ⚪ P4（待评估） | 需求待确认或远期规划 | 持续跟踪 |
