# Kimi Timer V1.0 — 桌面计时器

> 一款极简、精准、美观的 Windows 桌面计时器，基于 Electron 构建。

---

## 项目简介

Kimi Timer 是一款专为 Windows 设计的独立桌面计时器应用，支持**倒计时**和**正计时**两种模式，具备精准计时、窗口置顶、视觉与听觉提醒等功能。无需安装任何运行环境，双击即可运行。

## 核心功能

| 功能 | 说明 |
|------|------|
| 倒计时模式 | 自定义时/分/秒，倒计时归零自动提醒 |
| 正计时模式 | 从 0 开始累计计时，随时暂停/重置 |
| 精准计时 | 基于时间戳差值算法，最小化/后台运行不影响精度 |
| 窗口置顶 | 一键切换窗口始终显示在最前端 |
| 视觉提醒 | 倒计时结束时界面底色闪烁 3 秒 |
| 声音提醒 | Web Audio API 生成三段递进蜂鸣声 |
| 系统通知 | 调用 Windows 原生通知弹窗 |
| 极简界面 | 400×500 窗口，无顶部菜单栏，深色渐变风格 |

## 快速使用指南

### 1. 直接运行（推荐）

进入 `dist` 文件夹，双击运行：

```
Kimi Timer 1.0.0.exe
```

无需安装，即可使用。

### 2. 开发环境运行

```bash
npm start
```

窗口将弹出，即可开始计时。

### 操作说明

| 操作 | 说明 |
|------|------|
| 点击「倒计时/正计时」按钮 | 切换计时模式 |
| 时/分/秒输入框 | 仅倒计时模式下有效，设置目标时间 |
| 按 Enter 键 | 快速开始计时 |
| 点击「置顶」按钮 | 窗口始终保持在最前端 |
| 点击「开始」 | 启动计时器 |
| 点击「暂停」 | 暂停计时，保留已计时间 |
| 点击「重置」 | 清零，恢复初始状态 |

- 界面风格：深色渐变背景（`#1a1a2e` → `#16213e`），白色文字，渐变按钮

## 技术栈

- [Electron](https://www.electronjs.org/) — 桌面应用框架
- [Node.js](https://nodejs.org/) — 运行时环境
- HTML5 / CSS3 / JavaScript — 前端界面与逻辑
- [electron-builder](https://www.electron.build/) — 打包构建工具

## 项目目录

```
kimi-timer/
├── main.js              # 主进程（窗口创建、IPC、系统通知）
├── index.html           # 渲染进程（界面 + 计时逻辑 + 提醒系统）
├── package.json         # 项目配置与依赖
├── .npmrc               # 国内镜像源配置
├── dist/                # 打包输出目录
│   ├── Kimi Timer 1.0.0.exe   ← 便携版单文件可执行程序
│   └── win-unpacked/          ← 解压版目录
├── node_modules/        # 依赖目录
└── document/            # 项目文档目录
```
